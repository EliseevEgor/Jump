# Jump
Mario-style game

## Run

* <addr> cabal init
* <addr> cabal install
* <addr> cabal build
* <addr> cabal run Jump

## Controls 

* To start press s 
* To move left, right - arrows 
* To jump - space
* To exit - esc

## Game elements

* Black square - player
* Green cycles - monsters 
* Yellow cycles - coins 
* Tall green rectangle - flag
* Grey squares - boxes
* Red cycles - mushrooms
